String get_formatted_date_time ( )

Retrieves and formats the current date and time from the NTP server.

Gets the current date and time in ISO 8601 format.

This function fetches the current epoch time from the NTP server, breaks it down into its component year, month, day, hour, minute, and second, and then formats these components into a string according to the ISO 8601 standard. This format is widely used for data exchange and ensures that the timestamp is easily readable and unambiguous.

Returns
String A string representing the current date and time in ISO 8601 format.

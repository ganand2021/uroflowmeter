Connects to a WiFi network with the specified SSID and password.

Attempts to connect the ESP device to a WiFi network using the provided credentials.

Initiates a WiFi connection using the provided SSID and password, printing connection attempts and status to the serial console. Upon successful connection, it sets the isConnected flag to true and prints the device's assigned IP address.

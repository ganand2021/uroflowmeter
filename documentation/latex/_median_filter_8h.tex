\doxysubsection{Detailed Description}
Definition of the \doxylink{class_median_filter}{Median\+Filter} class for filtering values by computing their median. 

The \doxylink{class_median_filter}{Median\+Filter} class is useful for smoothing noisy data, particularly in the context of real-\/time sensor readings. It maintains a sliding window of the most recent values and computes the median of the values in this window, providing a filtered output that mitigates short-\/term fluctuations.

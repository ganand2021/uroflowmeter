\doxysubsection{Detailed Description}
Declarations of constants for AWS IoT connection. 

This file contains extern declarations for various constants required to connect to AWS IoT. It includes the endpoint, port, MQTT topics, device name, and certificates. These constants are defined in the corresponding source file ({\ttfamily \doxylink{secrets_8cpp}{secrets.\+cpp}}), which should be securely managed to protect sensitive information.

Connects to AWS IoT Core with the configured parameters and credentials. 

Initialize connection parameters for AWS IoT Core and establish a connection.

This function sets up the secure client with AWS\textquotesingle{}s public certificate, the device\textquotesingle{}s certificate, and private key. It attempts to connect to the MQTT server defined by AWS\+\_\+\+END\+\_\+\+POINT and AWS\+\_\+\+PORT. If the connection fails, it retries until the connection is successful.

Connects to a Wi\+Fi network with the specified SSID and password. 

Attempts to connect the ESP device to a Wi\+Fi network using the provided credentials.

Initiates a Wi\+Fi connection using the provided SSID and password, printing connection attempts and status to the serial console. Upon successful connection, it sets the {\ttfamily is\+Connected} flag to true and prints the device\textquotesingle{}s assigned IP address.

\doxysubsection{Detailed Description}
Implements a median filter for real-\/time data filtering. 

This class provides methods to add new values to the filter and to compute the median of the current window of values. It\textquotesingle{}s designed to work with floating point numbers, making it suitable for processing a wide range of data types.
